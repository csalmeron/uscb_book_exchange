About

The USCB Textbook Exchange website is comprised of a server-side web service and a client-side web service which allow students to have a facilitated experience while attempting to donate, exchange, and/or sell their books with other students.

About

The USCB Textbook Exchange website is comprised of a server-side web service that will consist of a database which will store user and book information in an organized manner and will handle all queries made by the users. Additionally, the web service will generate automated messages to expedite the transaction process. The client-side of the web service will allow the users to interact with the web service. In order to be able to use the web service the user must have access to a web browser that is capable of supporting the web service. Additionally, the user identity and student status will be verified through the use of their university email.
